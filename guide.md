## 웹개발 과정 ( 프론트엔드 + 백엔드 )

# 프로젝트를 위해 필요한 선택지

```
1. 서버선택
2. 백엔드 언어선택
3. 데이터베이스 선택
4. 아키텍트 만들기
5. 프론트엔드 선택( SPA, 링크방식 )
```



# "웹개발" 기술사항

```
1. OS 선택( window server, linux(추천) )
    - window
    - centos( 취업추천 ) 취업에서 사용하기 좋음 유료
    - ubuntu( 추천 ) 개인이 혼자서 하기 좋음 무료 오픈소스 - hanpixel
2. 웹서버( 웹언어를 해석할 수 있게 도와주는 역할이기도 하고 서버의 역할 )
    - apache ( 취업추천 )
    - nginx ( 떠오르는샛별 아파치보다 가볍고 빠른 ) - hanpixel
    - node ( 자바스크립트 ) - soket io 채팅프로그램 만들 수 있음 - hanpixel
3. 백엔드 언어 선택
    - php   - hanpixel
    - python ( 추천 ) - hanpixel
    - JAVA & JSP ( 취업추천 )
    - node  - hanpixel
    - go lang
4. 데이터베이스
    - oracle DB
    - mysql & mariadb ( 취업추천 ) - hanpixel
    - mongodb (NoSQL) 명령어가 없음
5. 개발방식
    - restful api ( 취업가장완전추천 ) - hanpixel 멀티디바이스 가능
    - MVC ( model - controller - view ) 할거임
6. 프론트엔드
    - HTMK, CSS, JS 은 필수
7. 프론트방식 선택
    - SPA 페이지(react - hanpixel, angular, vue 중 선택) 뷰가 조금 쉬움
    - 페이지이동 방식(링크)
8. ui / ux
    - 반응형
    - 하이브리드 ( 웹 + 모바일웹 )
    - 네이티브 ( 웹 + 앱 )
```
